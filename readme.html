Instructions of how to play Guess the number Game
1. At first you have to put the number 0 to 100, if you put the number 100 then it occur error(invalid number).

2. Computer choose randomly a number between 1 to 100, you have to guess this number, if you guess the number you win.

3. Suppose Computer generate a number (75), we don't know this number, we have to guess this number.

4. First you put your number (60) which is appears on your mind, then it shows "Your number is too low, guess highest number!!"

5. if you put 80, then it says "Your number is too high, guess lowest number!!".

6. After that if you put 75. Then it says you guess correct number.
